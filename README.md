# NewsApp

## Built With

- **React**

### News based on different categories

- Bussiness
- Entertainment
- Health
- Science
- Sports
- Technology

### Dependencies

- react-infinite-scroll-component
- react-top-loading-bar

# Installation

- `npm install`
- create `.env.local` file and create your api key from https://newsapi.org/
  - `REACT_APP_NEWS_API="apikey"` in `.env.local` file
- npm run start
